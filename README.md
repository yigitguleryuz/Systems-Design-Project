Beşiktaş App systems design project for System Analysis &amp; Design Class


# Beşiktaş JK Super App - System Analysis & Design Project

Bu proje, Beşiktaş JK taraftarları için kapsamlı bir mobil uygulama sistem analizidir.Mevcut Passolig sistemindeki sorunları çözmeyi ve kulüp ile taraftar arasındaki üçüncü partiyi kaldırarak doğrudan bir etkileşim kurmayı hedefledik.



## Proje Ekibi
Harun Enes Oğuz - Grup Lideri
Yiğit Güleryüz - Tasarım & Analiz
Kerem İnan- Analiz
Asil Gündüz- Pazarlama
Enes Dikmen- Tasarım

## Özellikler (Key Features)
Sistem analizi kapsamında tasarlanan temel fonksiyonlar:
Biletleme Sistemi
E-Ticaret Entegrasyonu
Stadum İçi Hizmetler
Taraftar Etkileşimi
Ceza Sistemi

## Sistem Tasarımı
Veri Akış Diyagramları
Veri Tabanı Yapısı
UI/UX Tasarımları (Mockups)



Not: Tasarımlar fikir vermesi açısından AI araçları ile oluşturulmuştur
